# Clone or download app

  git clone git@github.com:shalurai/Books.git   
  cd Books
  npm install
  npm start

# Clone or download server for this app

 [Server Link](https://github.com/shalurai/BookServer)

   git clone   git@github.com:shalurai/BookServer.git
   cd BookServer
   npm install
   node server.js
